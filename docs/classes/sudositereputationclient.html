Class SudoSiteReputationClient

This is the main class used for checking a site's reputation. Each instance of SudoSiteReputationClient will be configured with a ruleset from a ruleset provider to check a site's reputation against. To determine a site's reputation, you can call SudoSiteReputationClient.getSiteReputation.
